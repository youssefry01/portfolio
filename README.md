# Portfolio Website

This repository contains the source code for my personal portfolio website — a
full-stack developer portfolio built to showcase my projects, experience, and
technical background.

## Live Website
[View Portfolio](https://yousseframadan.vercel.app/)

## Features
- Light/dark theme with system-preference detection and persisted user choice
- Responsive layout across mobile, tablet, and desktop
- Scrollspy navigation that highlights the section currently in view
- Downloadable resume and direct links to GitHub, LinkedIn, and other profiles

## Tech Stack
- **Frontend:** React (Vite), Tailwind CSS
- **Routing:** React Router
- **Contact form:** EmailJS
- **Deployment:** Vercel
